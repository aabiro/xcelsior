## Overview

The Xcelsior TypeScript SDK is auto-generated from the OpenAPI specification using [Fern](https://buildwithfern.com). It provides type-safe access to all 175+ API endpoints.

## Installation

```bash
npm install @xcelsior-gpu/sdk
# or
yarn add @xcelsior-gpu/sdk
# or
pnpm add @xcelsior-gpu/sdk
```

## Quick Start

```typescript
import { XcelsiorApiClient, XcelsiorApiEnvironment } from "@xcelsior-gpu/sdk";

const client = new XcelsiorApiClient({
  environment: XcelsiorApiEnvironment.Production,
  headers: {
    Authorization: `Bearer ${process.env.XCELSIOR_API_KEY}`,
  },
});

// List all GPU hosts
const hosts = await client.hosts.list();
console.log(`${hosts.length} active hosts`);

// Submit an instance
const instance = await client.instances.create({
  name: "llama-inference",
  vramNeededGb: 24,
  priority: 5,
  tier: "sovereign",
});
console.log(`Instance ${instance.jobId} created`);
```

## Configuration

```typescript
import { XcelsiorApiClient, XcelsiorApiEnvironment } from "@xcelsior-gpu/sdk";

const client = new XcelsiorApiClient({
  // Environment (defaults to production)
  environment: XcelsiorApiEnvironment.Production, // "https://xcelsior.ca"

  // Authentication via headers
  headers: {
    Authorization: "Bearer your-api-key",
  },

  // Optional settings
  timeoutInSeconds: 60,  // request timeout
  maxRetries: 2,         // retry count (default: 2)
});
```

### Configuration Options

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `environment` | `string` | No | API base URL. Defaults to `https://xcelsior.ca` |
| `headers` | `Record<string, string>` | No | Additional headers (use for `Authorization`) |
| `timeoutInSeconds` | `number` | No | Request timeout in seconds |
| `maxRetries` | `number` | No | Number of retries on failure (default: 2) |
| `fetch` | `typeof fetch` | No | Custom fetch implementation |

## Usage Examples

### GPU Hosts

```typescript
// Register a host
await client.hosts.register({
  hostId: "my-gpu-01",
  ip: "100.64.0.5",
  gpuModel: "RTX 4090",
  vramGb: 24,
  costPerHour: 1.2,
  country: "CA",
  province: "ON",
});

// List hosts
const hosts = await client.hosts.list();

// Get a specific host
const host = await client.hosts.get("host-id-123");

// Remove a host
await client.hosts.remove("host-id-123");

// Get compute scores
const scores = await client.hosts.listComputeScores();
```

### Instance Management

```typescript
// Submit an instance (GPU job)
const instance = await client.instances.create({
  name: "training-run",
  vramNeededGb: 48,
  priority: 8,

// Check status
const status = await client.instances.get("job-id-123");

// List all instances

// Cancel an instance
await client.instances.cancel("job-id-123");

// Get logs
const logs = await client.instances.getLogs("job-id-123");
```

## Error Handling

```typescript
import { XcelsiorApiClient, XcelsiorApiError, XcelsiorApiTimeoutError } from "@xcelsior-gpu/sdk";

try {
  const instance = await client.instances.create({ name: "my-job", vramNeededGb: 24 });
} catch (error) {
  if (error instanceof XcelsiorApiTimeoutError) {
    console.error("Request timed out");
  } else if (error instanceof XcelsiorApiError) {
    console.error(`API Error ${error.statusCode}: ${error.message}`);
    console.error("Response body:", error.body);
  } else {
    throw error;
  }
}
```

## TypeScript Support

The SDK is fully typed. All request/response objects have TypeScript interfaces:

```typescript
import { XcelsiorApi } from "@xcelsior-gpu/sdk";

// Access types via the XcelsiorApi namespace
type ValidationError = XcelsiorApi.ValidationError;
type HttpValidationError = XcelsiorApi.HttpValidationError;
```

## Passthrough Requests

For endpoints not yet covered by the SDK, use the built-in `fetch` method which inherits your auth, retry, and logging configuration:

```typescript
const response = await client.fetch("/api/some-new-endpoint", {
  method: "POST",
  body: JSON.stringify({ key: "value" }),
  headers: { "Content-Type": "application/json" },
});
const data = await response.json();
```
